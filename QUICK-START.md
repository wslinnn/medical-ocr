# 🚀 快速启动指南

## 一、环境准备

### 1. 安装 Node.js
确保已安装 Node.js v16 或更高版本

```bash
node --version
```

### 2. 安装依赖

```bash
cd D:\build-electron\electron-web-wrapper
npm install
```

---

## 二、准备图标（可选）

在 `assets/` 目录下放置以下图标：

1. **icon.png** - 应用图标（512x512px）
2. **tray-icon.png** - 托盘图标（16x16px 或 32x32px）

*没有图标也可以运行，但会使用默认图标*

---

## 三、配置 API Token

### 方式1: 启动后配置
1. 启动应用后，在首页输入 API Token
2. 点击"保存"按钮

### 方式2: 直接修改
打开应用后，在 Token 输入框填入：
```
f29f887a3d4c22faa87154df9da5f13e83112c74
```

---

## 四、启动应用

### 开发模式（推荐首次使用）
```bash
npm run dev
```

### 生产模式
```bash
npm start
```

---

## 五、基本使用流程

```
1. 输入 API Token → 保存
   ↓
2. 选择文件/文件夹
   ↓
3. 点击"开始全部识别"
   ↓
4. 等待识别完成
   ↓
5. 查看"记录"视图
   ↓
6. 对比原图确认结果
   ↓
7. 编辑并标记状态
   ↓
8. 导出 Excel
```

---

## 六、常用操作

### 批量处理
- 点击"选择文件夹"一次性选择整个病历文件夹
- 系统会自动识别所有支持的文件

### 快速审核
- 切换到"对比"视图
- 使用方向键快速切换记录
- 点击"标记已审核"确认

### 批量导出
- 在"记录"视图筛选需要的记录
- 点击复选框选择
- 点击"导出选中"

### 全局快捷键
- `Ctrl+Shift+O`: 最小化后快速打开并选择文件
- `Ctrl+Shift+E`: 快速导出所有数据

## 七、打包分发

### 构建 Windows 安装包
```bash
npm run build:win
```

### 构建便携版
```bash
npm run build:portable
```

构建完成后，在 `dist/` 目录找到安装包

---

## 八、故障排除

### 应用无法启动
```bash
# 清除缓存重新安装
rm -rf node_modules
npm install
npm run dev
```

### Token 无效
- 检查 Token 是否正确
- 确认网络连接
- 尝试重新获取 Token

### 识别失败
- 确认文件格式正确（JPG/PNG/PDF）
- 检查文件大小（建议<10MB）
- 确保图片清晰可辨

### 找不到数据
- 检查浏览器控制台（开发模式按F12）
- 数据存储在 LocalStorage 中
- 建议定期导出备份
