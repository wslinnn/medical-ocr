# 图标资源说明

本应用需要以下图标资源：

## 必需图标

### 1. icon.png
- **用途**: 应用程序主图标
- **尺寸**: 512x512px (PNG格式，支持透明背景)
- **说明**: 用于应用窗口标题栏、任务栏等

### 2. tray-icon.png
- **用途**: 系统托盘图标
- **尺寸**: 16x16px 或 32x32px (PNG格式，支持透明背景)
- **说明**: 最小化后显示在系统托盘的图标

### 3. tray-icon@2x.png (可选)
- **用途**: 高DPI系统托盘图标
- **尺寸**: 32x32px 或 64x64px

## 图标设计建议

1. **主题**: 医疗健康相关
2. **颜色**: 使用主色调 #0891B2 (医疗蓝绿)
3. **元素**: 可包含文档/OCR相关的视觉元素
4. **风格**: 简洁、专业、易识别

## 临时解决方案

如果暂时没有准备图标，可以使用以下方法：

### 方法1: 使用在线工具生成
- 访问 https://favicon.io 或类似网站
- 生成简单的文字图标作为临时使用

### 方法2: 使用Electron默认图标
- 不提供图标文件时，Electron会使用默认图标
- 应用仍然可以正常运行，只是外观不够专业

### 方法3: 从现有文件转换
如果有其他格式的图标，可以使用在线工具转换：
- https://convertico.com/
- https://www.aconvert.com/cn/image/png-to-ico.html

## 图标文件放置位置

```
assets/
├── icon.png           (主图标，512x512)
├── tray-icon.png      (托盘图标，16x16 或 32x32)
└── README.md          (本说明文件)
```

## Windows 可执行文件图标

如果需要为 .exe 文件设置图标，需要：
1. 准备 .ico 格式文件
2. 在 build-config.js 中配置
3. 或使用 electron-builder 的 win.icon 配置

## 注意事项

- 图标文件必须是 PNG 格式
- 建议使用透明背景
- 确保图标在小尺寸下仍然清晰可辨
