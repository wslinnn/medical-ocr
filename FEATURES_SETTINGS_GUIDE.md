## ✨ 新增功能

### 1. 右上角设置按钮

**位置**: 页面标题右侧
**功能**: 打开API配置设置模态框

**特性**:
- 齿轮图标，直观易识别
- 悬停时有背景变化反馈
- 点击打开设置模态框

---

### 2. 网络状态显示

**位置**: 设置按钮旁边
**功能**: 实时显示网络连接状态

**状态**:
- 🟢 **已连接** (绿色) - 网络正常
- 🔴 **已断开** (红色) - 网络断开

**自动更新**:
- 页面加载时自动检测
- 网络状态变化时自动更新
- 断开/恢复时显示Toast提示

---

### 3. 设置模态框

**打开方式**: 点击右上角设置按钮

**功能**:
- API Token 配置
- Token 显示/隐藏切换
- Token 状态指示
- 帮助信息

**特点**:
- 密码输入框保护Token安全
- 一键显示/隐藏Token
- 实时验证Token格式
- 友好的帮助提示

---

### 4. 新用户引导系统

**触发条件**:
- 首次使用系统（localStorage无`has_seen_guide`标记）
- API Token未配置时

**引导流程**:

#### 步骤1: 欢迎
- 系统介绍
- 主要功能展示（批量上传、智能识别、质量控制）
- 可选择"跳过引导"或"下一步"

#### 步骤2: 配置Token
- API Token输入
- 重要提示和安全说明
- 格式验证
- 保存后自动进入下一步

#### 步骤3: 开始使用
- 配置完成提示
- 使用指南（3个快速上手步骤）
- "开始使用"按钮完成引导

**引导状态持久化**:
- `localStorage.setItem('has_seen_guide', 'true')`
- 完成引导后不再自动显示
- 可通过设置按钮手动配置Token

## 🎯 使用流程

### 首次使用流程

1. **打开应用** → 自动检测到未配置Token
2. **显示引导** → 3步骤引导流程
3. **配置Token** → 在引导中输入API Token
4. **完成引导** → 进入主界面开始使用

### 已配置用户

1. **打开应用** → 直接进入主界面
2. **需要修改配置** → 点击右上角设置按钮
3. **在设置中修改** → 保存Token

### 跳过引导

- 首次使用时可点击"跳过引导"
- 系统提示：可随时点击设置按钮配置
- 设置按钮在右上角，齿轮图标

```
│   ├── showGuideModal()
│   ├── hideGuideModal()

│   ├── nextGuideStep()
│   ├── prevGuideStep()
```

### 状态管理

```javascript
// localStorage keys
'aistudio_token'        // API Token

// state 对象
```

- 使用 `pro-focus-manager.js` 提供的焦点管理函数
- 模态框打开时自动聚焦第一个输入框
- 模态框关闭时恢复焦点到触发元素

- **悬停**: 背景变为浅灰色，边框变为灰色

### 引导交互
- **步骤切换**: 动画过渡，步骤指示器更新
- **输入验证**: 实时检查Token长度
- **完成引导**: 成功提示并关闭模态框

### Token显示/隐藏
- **默认**: 密码模式 `•••`
- **切换**: 点击眼睛图标
- **图标变化**:
  - 隐藏: 斜线眼睛图标
  - 显示: 正常眼睛图标

### 5. 页面布局重构

**更新日期**: 2026-03-07

**改动**: 合并文件上传和识别记录页面，优化工作流程

**新布局结构**:
```
┌─────────────────────────────────────────────────────────┐
│                    文件处理 (Tab)                         │
├──────────────────────┬──────────────────────────────────┤
│   左侧 (35%)         │        右侧 (65%)                 │
├──────────────────────┼──────────────────────────────────┤
│   上传病历文件       │    ┌────────────────────────┐    │
│   ┌──────────────┐   │    │ 统计卡 (4列)           │    │
│   │ 选择文件夹    │   │    │ 今日|待审|已审|需复核  │    │
│   │ 选择文件      │   │    └────────────────────────┘    │
│   │ 拖拽区域      │   │    ┌────────────────────────┐    │
│   └──────────────┘   │    │ 搜索 & 状态筛选         │    │
│                      │    └────────────────────────┘    │
│   文件队列           │    ┌────────────────────────┐    │
│   ┌──────────────┐   │    │ 导出全部 | 数据管理     │    │
│   │ 文件列表      │   │    └────────────────────────┘    │
│   │ 进度条        │   │    ┌────────────────────────┐    │
│   │ 开始识别      │   │    │ 记录表格 (可批量选择)   │    │
│   └──────────────┘   │    └────────────────────────┘    │
│                      │    ┌────────────────────────┐    │
│                      │    │ 分页控件                │    │
│                      │    └────────────────────────┘    │
└──────────────────────┴──────────────────────────────────┘
```

**关键变化**:
1. **标签页简化**: 从3个标签减少到2个（文件处理、对比审核）
2. **统一视图**: 上传和记录在同一页面，无需切换
3. **高效布局**: 左侧专注于文件操作，右侧专注于数据管理
4. **保留功能**: 所有原有功能保持完整（批量操作、导出、搜索等）

**修改的文件**:
- `medical-ocr-pro.html` - 重构view-upload布局，删除view-records
- `js/pro-tabs.js` - 更新switchTab逻辑，记录渲染在upload tab触发
- `js/pro-events-tabs.js` - 移除tab-records事件处理器
